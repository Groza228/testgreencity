# GreenCity UI Tests (Python + Selenium)

## Опис проєкту

Цей проєкт містить автоматизовані UI-тести для сайту GreenCity.
Тести реалізовані з використанням Python, Selenium WebDriver та unittest без сторонніх фреймворків.

## Посилання на сторінку тестування

https://www.greencity.cx.ua/#/greenCity/events

## Встановлення

```bash
pip install -r requirements.txt
```

## Запуск тестів

```bash
python -m unittest discover tests
```

## Реалізовані тести

* Реєстрація користувача
* Перевірка сторінки подій
* Фільтрація новин
